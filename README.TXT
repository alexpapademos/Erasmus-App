🧾 Περίληψη Εργασίας HTML – Erasmus+ App

Η παρούσα εργασία αφορά την ανάπτυξη ενός ιστότοπου για την υποστήριξη αιτήσεων στο πρόγραμμα Erasmus+. Ο ιστότοπος παρέχει τις βασικές λειτουργίες για την ενημέρωση των φοιτητών και την υποβολή των αιτήσεών τους, με λειτουργίες τόσο στα Ελληνικά όσο και στα Αγγλικά.
📄 Περιλαμβάνει:

    index.html: Αρχική σελίδα με εισαγωγικό κείμενο και εικόνα.

    more.html: Πληροφορίες για το πρόγραμμα Erasmus+, σύνδεσμοι προς συνεργαζόμενα πανεπιστήμια και βίντεο.

    reqs.html: Πίνακας με τις ελάχιστες απαιτήσεις για συμμετοχή στο πρόγραμμα και φόρμα γρήγορου ελέγχου.

    application.html: Κύρια φόρμα για την υποβολή αίτησης με πεδία εισαγωγής, επιλογές, και δυνατότητα ανέβασμα αρχείων.

    sign-up.html: Φόρμα για τη δημιουργία λογαριασμού νέου χρήστη.

    login.html: Φόρμα εισόδου με δυνατότητα "Να με θυμάσαι" για αυτόματη είσοδο στο μέλλον.

🎨 Σχεδίαση:

    Χρήση CSS3 για την ανάπτυξη ενός καθαρού και επαγγελματικού σχεδιασμού χωρίς χρήση εξωτερικών framework.

    Responsive design, ώστε ο ιστότοπος να είναι συμβατός και λειτουργικός σε όλες τις συσκευές (desktop, tablet, κινητό).

    Χρήση base.css για κοινό στυλ (μενού, layout, fonts) και forms.css για τη στυλιστική διαμόρφωση των φορμών, όπως τα inputs, radio buttons, checkboxes, και τα κουμπιά.

🧪 Προαιρετικά (πρόσθετα χαρακτηριστικά):

    JS Validation:

        Επαλήθευση της συμφωνίας των κωδικών κατά την εγγραφή χρήστη.

        Έλεγχος της ορθότητας του email κατά την εγγραφή.

    Διαχείριση γλώσσας:

        Υποστήριξη για αλλαγή γλώσσας μεταξύ Ελληνικών και Αγγλικών, με τη δυνατότητα αποθήκευσης της επιλογής γλώσσας στον τοπικό αποθηκευτικό χώρο (localStorage), ώστε να παραμένει η επιλογή γλώσσας ακόμα και αν ο χρήστης μεταβεί σε άλλες σελίδες ή ξαναεπιστρέψει στον ιστότοπο.

    Αποθήκευση δεδομένων:

        Δυνατότητα αποθήκευσης δεδομένων στις φόρμες, είτε σε τοπικό αρχείο JSON για επίδειξη, είτε με console.log() για debug και δοκιμές.

🧾 Τεχνολογίες:

    HTML5: Δομή σελίδας και περιεχόμενο.

    CSS3: Εξωτερικά φύλλα στυλ για την εμφάνιση και τον σχεδιασμό.

    JavaScript (προαιρετικά): Χρησιμοποιείται για έλεγχο εισόδων, validation, αλλαγή γλώσσας, και dynamic content management.

🔧 Βελτιώσεις και Υλοποιήσεις:

    Διαχείριση γλώσσας: Αντικαταστήσαμε τη γλώσσα δυναμικά με τη χρήση localStorage, για να αποθηκεύεται η προτίμηση γλώσσας του χρήστη και να εμφανίζεται στις επόμενες σελίδες χωρίς να την χρειάζεται να την επιλέγει ξανά.

    Αλλαγή στο περιβάλλον εργασίας: Χρησιμοποιήσαμε responsive design για να εξασφαλίσουμε ότι η εφαρμογή λειτουργεί σωστά σε όλες τις συσκευές (desktop, tablet, κινητό).

    Επαλήθευση δεδομένων με JavaScript: Εφαρμόσαμε λειτουργίες για να εξασφαλίσουμε ότι οι χρήστες συμπληρώνουν σωστά τα πεδία (π.χ. συμφωνία κωδικών κατά την εγγραφή και ορθότητα email).
